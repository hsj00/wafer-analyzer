
# 🔬 웨이퍼 맵 분석기 (Wafer Map Analyzer) v3.0

반도체 공정 데이터를 웹 브라우저에서 즉시 분석할 수 있는 Streamlit 기반 애플리케이션입니다.

CSV 또는 Excel 파일을 불러오거나 **데이터를 직접 입력**하여 **웨이퍼 맵 시각화 · 다중 웨이퍼 비교 · GPC 분석 · 결함 오버레이 · ML 이상 탐지 · Excel 보고서 생성**을 한 화면에서 처리합니다.

## 1. 이런 분께 필요합니다

* 박막 두께, 식각 깊이, 면저항 등 **웨이퍼 전면 분포 데이터**를 시각화하고 싶은 공정 엔지니어
* ALD 공정의 **GPC(Growth Per Cycle)** 균일도를 분석하고 싶은 연구자
* 결함 검사 장비 데이터를 공정 측정값과 **겹쳐서 보고 싶은** QC 담당자
* 여러 웨이퍼를 한꺼번에 비교하거나 **이상 웨이퍼를 자동으로 탐지**하고 싶은 엔지니어
* 데이터 파일 없이 **측정값을 직접 입력**해서 빠르게 맵을 확인하고 싶은 엔지니어

## 2. v3.0 주요 변경사항

### ① 비교 모드 → 웨이퍼 맵 탭 통합

기존 사이드바의 비교 모드 토글이 제거되고, **웨이퍼 맵 탭 내부에 서브탭**으로 통합되었습니다.

| v2.0                                     | v3.0                                                   |
| ---------------------------------------- | ------------------------------------------------------ |
| 사이드바 `비교 모드 활성화`토글        | 웨이퍼 맵 탭 →`🔬 단일 분석`/`🔀 비교 분석`서브탭 |
| 비교 모드 ON 시 모든 탭이 카드 뷰로 교체 | 단일 분석과 비교 분석을 자유롭게 전환                  |
| 사이드바에 데이터셋 관리 UI              | 비교 서브탭 내부에 데이터셋 관리 UI                    |

### ② 수동 입력 모드 추가

**데이터 파일 없이도 앱을 사용할 수 있습니다.** 단일 분석 서브탭에서 "✏️ 수동 입력"을 선택하면 빈 테이블이 표시되며, X/Y 좌표와 측정값을 직접 입력하거나 스프레드시트에서 붙여넣기(Ctrl+V)할 수 있습니다. 유효 포인트가 3개 이상이면 차트가 자동 생성됩니다.

### ③ 다중 파라미터 탭 제거 → 비교 서브탭으로 통합

기존 `📐 다중 파라미터` 탭이 제거되었습니다. 같은 파일의 여러 측정 컬럼을 비교하려면 **비교 서브탭에서 같은 파일을 Data 컬럼만 다르게 여러 번 추가**하면 됩니다.

## 3. 전체 기능 흐름

```
CSV / Excel 파일  또는  수동 입력(직접 타이핑 / 붙여넣기)
        │
        ▼
① 📊 웨이퍼 맵 ─┬─ 🔬 단일 분석  ← 파일 로드 또는 수동 입력으로 시작
                 └─ 🔀 비교 분석  ← 여러 웨이퍼/파라미터 나란히 비교
        │
        ├──► ② 🔍 결함 오버레이   별도 결함 CSV를 웨이퍼 맵 위에 겹쳐 표시
        │
        ├──► ③ ⚗️ GPC 분석        두께 ÷ 사이클 수, 반경별 균일도 프로파일
        │
        ├──► ④ 📄 보고서 생성     통계 + 차트 이미지 + 원본 데이터 → xlsx 다운로드
        │
        └──► ⑤ 🤖 ML 이상 탐지   여러 웨이퍼를 PCA + IsolationForest로 자동 분류
```

> **탭 ②~④**는 단일 분석에서 데이터를 먼저 로드한 후 사용하세요.
>
> **탭 ⑤**는 독립된 데이터셋 패널이 있어 탭 안에서 직접 파일을 추가할 수 있습니다.
>
> 비교 서브탭에서 추가한 데이터셋은 ML 탭에서 자동으로 활용됩니다.

## 4. 프로젝트 구조

```
wafer_analysis/
│
├── app.py                   # 메인 앱 진입점 — 이 파일을 실행합니다
├── folder_picker_helper.py  # OS 네이티브 폴더 선택 보조 스크립트 (자동 호출됨)
│
└── modules/
    ├── __init__.py          # 모듈 안전 로드 관리 (수정 불필요)
    ├── defect_overlay.py    # 탭 ② 결함 오버레이
    ├── gpc.py               # 탭 ③ GPC 분석
    ├── report.py            # 탭 ④ Excel 보고서 생성
    └── ml_anomaly.py        # 탭 ⑤ ML 기반 이상 탐지
```

> **⚠️ 주의:** `modules/` 폴더와 그 안의 파일이 모두 있어야 합니다.
>
> `app.py`만 단독으로 있으면 탭 ②~⑤가 전부 `⚠️` 상태로 비활성화됩니다.
>
> v3.0에서 `multi_param.py`는 삭제되었습니다. 기존 파일이 남아있어도 무시됩니다.

## 5. 설치 및 실행

### 5-1. Python 버전 확인

Python **3.10 이상**이 필요합니다.

```bash
python --version
# Python 3.10.x 이상이어야 합니다
```

### 5-2. 가상환경 생성 (권장)

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 5-3. 패키지 설치

`requirements.txt`가 있다면:

```bash
pip install -r requirements.txt
```

없다면 직접 설치:

```bash
pip install streamlit pandas numpy plotly scipy openpyxl scikit-learn kaleido
```

| 패키지       | 용도                                | 없으면                  |
| ------------ | ----------------------------------- | ----------------------- |
| streamlit    | 웹 UI 프레임워크                    | 앱 실행 불가            |
| pandas       | CSV/Excel 읽기 및 데이터 처리       | 앱 실행 불가            |
| numpy        | 수치 연산                           | 앱 실행 불가            |
| plotly       | 인터랙티브 차트                     | 앱 실행 불가            |
| scipy        | 웨이퍼 격자 보간 (`griddata`)     | 앱 실행 불가            |
| openpyxl     | Excel 파일 읽기/쓰기                | 앱 실행 불가            |
| scikit-learn | ML 이상 탐지 (PCA, IsolationForest) | 🤖 ML 탭만 비활성화     |
| kaleido      | 차트 → PNG 변환 (보고서 이미지용)  | 이미지 없이 보고서 생성 |

> `scikit-learn`과 `kaleido`는 선택 패키지입니다. 없어도 나머지 탭은 정상 작동합니다.

### 5-4. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501`이 자동으로 열립니다.

### 5-5. 처음 실행 시

데이터 파일이 없어도 바로 시작할 수 있습니다. 두 가지 방법이 있습니다:

**방법 A — 샘플 데이터 생성:**

1. 앱 실행 후 왼쪽 사이드바에서 **"🎯 샘플 5개 생성"** 버튼을 클릭합니다.
2. `./wafer_data/` 폴더에 `wafer_01.csv` ~ `wafer_05.csv`가 자동 생성됩니다.

**방법 B — 수동 입력 (v3.0 신규):**

1. 웨이퍼 맵 탭 → 단일 분석 서브탭에서 **"✏️ 수동 입력"**을 선택합니다.
2. 빈 테이블에 X, Y, Data 값을 직접 입력하거나 스프레드시트에서 붙여넣습니다.
3. 유효 포인트 3개 이상이면 즉시 웨이퍼 맵이 생성됩니다.

## 6. 입력 데이터 형식

### 6-1. 기본 웨이퍼 맵 데이터 (CSV / Excel)

X·Y 좌표와 측정값 컬럼이 포함된 파일이면 됩니다.

| x     | y     | data  |
| ----- | ----- | ----- |
| 0.0   | 100.0 | 512.3 |
| -50.0 | 86.6  | 498.7 |
| -86.6 | 50.0  | 503.1 |

* **컬럼명 자동 인식:** `x`, `y`, `data` (대소문자 무관). 다른 이름이라도 사이드바 **🔗 컬럼 매핑**에서 직접 지정할 수 있습니다.
* **단위:** mm 기준. 200 mm 웨이퍼 → 반경 약 100 mm, 300 mm 웨이퍼 → 반경 약 150 mm
* **최소 포인트 수:** 보간 품질을 위해 10개 이상 권장 (수동 입력 시 최소 3개)

### 6-2. 수동 입력 데이터 (v3.0 신규)

파일 없이 웨이퍼 맵 탭에서 직접 입력합니다.

* 단일 분석 서브탭에서 "✏️ 수동 입력" 선택
* 20행 빈 테이블(x, y, data)이 표시됩니다
* 셀에 직접 타이핑하거나, 스프레드시트에서 범위를 복사해 **Ctrl+V**로 붙여넣기
* 행 추가/삭제 가능 (테이블 하단 `+` 버튼)
* NaN 행은 자동 제외, 유효 포인트 3개 이상이면 차트 자동 생성
* 🗑️ 초기화 버튼으로 빈 테이블로 리셋 가능

### 6-3. GPC 분석용 데이터

두께 컬럼과 사이클 수 컬럼이 추가로 필요합니다.

| x     | y     | thickness_nm | n_cycles |
| ----- | ----- | ------------ | -------- |
| 0.0   | 100.0 | 51.2         | 100      |
| -50.0 | 86.6  | 49.8         | 100      |

* 모든 포인트의 사이클 수가 동일하다면 컬럼 없이 탭에서 숫자를 직접 입력해도 됩니다.

### 6-4. 결함 오버레이용 데이터 (별도 파일)

웨이퍼 맵과 같은 데이터 폴더에 별도 파일로 준비합니다.

| x     | y     | class    | size | description   |
| ----- | ----- | -------- | ---- | ------------- |
| 10.5  | -20.3 | Particle | 5.0  | 큰 입자       |
| -33.2 | 41.8  | Scratch  | 12.0 | 선형 스크래치 |

* **결함 분류 컬럼:** `class`, `type`, `category`, `defecttype`, `label` 중 하나를 자동 인식합니다.
* `size`와 `description`은 선택 컬럼입니다. 없어도 동작합니다.
* **좌표 단위가 다른 경우** (예: 결함 파일은 μm, 웨이퍼 파일은 mm): 탭 내 단위 변환 옵션으로 보정합니다.

## 8. 탭별 사용 방법

### 탭 ① — 📊 웨이퍼 맵

**모든 탭의 시작점**입니다.

#### 🔬 단일 분석 서브탭

**데이터 소스 선택**

| 모드           | 설명                                             |
| -------------- | ------------------------------------------------ |
| 📁 파일 데이터 | 사이드바에서 선택한 CSV/Excel 파일을 분석 (기본) |
| ✏️ 수동 입력 | 빈 테이블에 직접 데이터를 입력하거나 붙여넣기    |

> 데이터 폴더에 파일이 없으면 "✏️ 수동 입력"만 표시됩니다.

**제공 차트**

| 차트       | 설명                                               |
| ---------- | -------------------------------------------------- |
| 2D Heatmap | 측정값을 색상 그라데이션으로 나타낸 기본 웨이퍼 맵 |
| Contour    | 등고선으로 분포 경계를 시각화                      |
| Line Scan  | 지정한 각도 방향의 단면 프로파일                   |
| 3D Surface | 측정값의 높낮이를 3차원으로 표현                   |

**통계:** Mean · Maximum · Minimum · Std Dev · Uniformity(%) · Range · 측정 포인트 수

**Raw Data 편집:** 탭 하단의 데이터 테이블에서 셀을 직접 수정하면 차트가 즉시 갱신됩니다. 수정한 데이터는 **📥 CSV 다운로드** 버튼으로 저장할 수 있습니다.

#### 🔀 비교 분석 서브탭

여러 웨이퍼 또는 같은 웨이퍼의 여러 파라미터를 **나란히 비교**합니다.

> **v2.0의 비교 모드 + 다중 파라미터 기능을 통합**한 서브탭입니다.

**사용 순서**

1. 비교 분석 서브탭으로 이동합니다.
2. **➕ 데이터셋 추가** 를 펼쳐 파일·시트·X/Y/Data 컬럼을 지정합니다.
3. 데이터셋을 2개 이상 추가하면 카드 형태로 비교 뷰가 표시됩니다.
4. 각 카드에 **Heatmap + Contour + 통계**가 표시됩니다.

**주요 기능**

* **🔒 컬러 스케일 통일** 옵션으로 모든 카드의 색상 범위를 맞출 수 있습니다.
* **행당 맵 개수**를 2/3/4개 중 선택하여 레이아웃을 조절합니다.
* **같은 파일에서 Data 컬럼만 다르게** 여러 번 추가하면 다중 파라미터 비교가 가능합니다.

**다중 파라미터 비교 예시:**

같은 `wafer_01.xlsx`에서:

* 데이터셋 1: X=x, Y=y, Data=**thickness**
* 데이터셋 2: X=x, Y=y, Data=**sheet_resistance**
* 데이터셋 3: X=x, Y=y, Data=**stress**

→ 3개 카드가 나란히 표시되어 같은 웨이퍼의 세 가지 물성을 한눈에 비교할 수 있습니다.

---

### 탭 ② — 🔍 결함 오버레이

**웨이퍼 맵 위에 결함 좌표를 겹쳐** 공정 데이터와 결함의 공간적 상관관계를 분석합니다.

**사용 순서**

1. 결함 CSV/Excel 파일을 데이터 폴더에 저장합니다.
2. 🔍 결함 오버레이 탭에서 결함 파일을 선택합니다.
3. 표시할 결함 클래스를 선택합니다.
4. 결함 좌표 단위가 웨이퍼 데이터와 다른 경우 단위 변환 옵션을 설정합니다.

**주요 기능**

* Heatmap 또는 Contour 위에 결함 산점도 오버레이
* 결함 클래스별 색상·마커 자동 구분 (최대 24종)
* 좌표 단위 변환 (mm / cm / m / inch)
* 웨이퍼 외부 결함 표시/숨김

---

### 탭 ③ — ⚗️ GPC 분석

ALD(Atomic Layer Deposition) 공정의 **GPC(Growth Per Cycle)** 값을 계산하고 공간 분포를 분석합니다.

> **GPC = 두께(nm) ÷ 사이클 수**

**사이클 수 입력 방식**

| 모드        | 설명                                                      |
| ----------- | --------------------------------------------------------- |
| 고정 사이클 | 모든 측정 포인트에 동일한 사이클 수 적용 (숫자 직접 입력) |
| 컬럼 사이클 | 파일 내 사이클 수 컬럼 값을 포인트별로 적용               |

**사용 순서**

1. 단일 분석에서 두께 데이터가 포함된 파일을 로드합니다.
2. ⚗️ GPC 탭으로 이동합니다.
3. 두께 컬럼과 사이클 모드를 선택합니다.
4. GPC Heatmap과 반경별 프로파일이 자동 생성됩니다.

**제공 결과:** GPC Heatmap · 반경별 GPC 프로파일 · Center / Mid / Edge 구역별 통계 및 Center-Edge 균일도

---

### 탭 ④ — 📄 보고서 생성

분석 결과를 **Excel(.xlsx) 파일**로 내보냅니다.

**Excel 시트 구성**

| 시트       | 내용                                                      |
| ---------- | --------------------------------------------------------- |
| Summary    | 파일명, 생성 시각, 통계 요약 (Mean, Std, Uniformity 등)   |
| Statistics | 상세 통계 데이터                                          |
| Maps       | Heatmap · Contour · Line Scan · 3D Surface 차트 이미지 |
| Raw Data   | 원본 측정 데이터 (최대 5,000행)                           |
| GPC        | GPC 분석 결과 및 차트 (GPC 탭 실행 후에만 포함됨)         |

**사용 순서**

1. 단일 분석에서 데이터를 로드합니다.
2. (선택) GPC 시트를 포함하려면 GPC 탭에서 분석을 먼저 실행합니다.
3. 📄 보고서 탭으로 이동합니다.
4. 포함할 항목(차트 이미지 / 원본 데이터 / GPC)을 체크합니다.
5. **Excel 보고서 생성** 버튼 클릭 → 브라우저에서 즉시 다운로드됩니다.

> 차트 이미지 포함 기능은 `kaleido` 패키지가 필요합니다.

### 탭 ⑤ — 🤖 ML 이상 탐지

**PCA + Isolation Forest**로 여러 웨이퍼 중 이상 패턴을 자동 탐지합니다.

최소 3개 이상의 웨이퍼 데이터가 필요합니다.

**사용 순서**

1. 🤖 ML 이상 탐지 탭으로 이동합니다.
2. **📋 분석 데이터셋** 패널에서 웨이퍼를 추가합니다.| 방법                       | 설명                                             |
   | -------------------------- | ------------------------------------------------ |
   | **➕ 데이터셋 추가** | 파일·시트·컬럼 선택 후 추가                    |
   | **🔄 앱 동기화**     | 비교 서브탭에서 추가한 데이터셋을 한 번에 가져옴 |
   | **✕ 버튼**          | 목록에서 개별 데이터셋 제거                      |
   | **🗑️ 전체 초기화** | 목록과 분석 결과를 모두 초기화                   |
3. 3개 이상 등록되면 파라미터를 설정합니다.| 파라미터                  | 기본값 | 설명                                |
   | ------------------------- | ------ | ----------------------------------- |
   | 이상 비율 (Contamination) | 10%    | 전체 웨이퍼 중 이상으로 간주할 비율 |
   | 특성 추출 해상도          | 40     | 낮을수록 빠름 / 높을수록 정밀       |
4. **🤖 이상 탐지 실행** 버튼을 클릭합니다.

**결과 화면**

* **PCA 산점도:** 이상 웨이퍼(빨간 점)와 정상 웨이퍼(파란 점)
* **이상 점수 바 차트:** 웨이퍼별 이상 점수 순위
* **결과 테이블:** 이상 점수 + 패턴 분류
* **이상 웨이퍼 Heatmap 미리보기**

**이상 패턴 분류**

| 패턴             | 특징               | 공정 원인 예시                       |
| ---------------- | ------------------ | ------------------------------------ |
| Ring             | 도넛형 방사 분포   | 가스 흐름 중심 집중 또는 확산 불균일 |
| Edge Degradation | 가장자리 두께 저하 | Edge exclusion, 로딩 효과            |
| X/Y-Gradient     | 방향성 두께 구배   | 기판 기울기, 가스 방향성             |
| Hotspot          | 국소 이상 피크     | 파티클, 스크래치, 측정 오류          |
| Global Shift     | 전체 수준 편차     | 레시피 변경, 공정 드리프트           |
| Normal           | 정상               | —                                   |

> `scikit-learn` 패키지가 필요합니다: `pip install scikit-learn`

## 9. 사용 예시

### 예시 A: 처음 실행 — 수동 입력으로 즉시 시작

```
1. streamlit run app.py
2. 웨이퍼 맵 탭 → 단일 분석 → ✏️ 수동 입력 선택
3. 빈 테이블에 좌표와 측정값을 직접 입력 (또는 엑셀에서 Ctrl+C → Ctrl+V)
4. 3개 이상 입력하면 Heatmap, Contour, 통계 자동 생성
```

### 예시 B: 샘플 데이터로 전체 기능 체험

```
1. streamlit run app.py
2. 사이드바 → 🎯 샘플 5개 생성 클릭
3. 파일 목록에서 wafer_01.csv 선택
4. 단일 분석 서브탭 → Heatmap, Contour, 통계 확인
5. 비교 분석 서브탭 → ➕ 데이터셋 추가로 5개 파일 모두 등록 → 비교
6. 🤖 ML 탭 → 🔄 앱 동기화 → 이상 탐지 실행
```

### 예시 C: 다중 파라미터 비교

```
1. 여러 측정 컬럼(thickness, rs, stress 등)이 있는 Excel 파일 준비
2. 비교 분석 서브탭 이동
3. ➕ 데이터셋 추가: 같은 파일에서 Data 컬럼만 다르게 3개 추가
4. 🔒 컬러 스케일 통일 체크 → 절대값 비교 가능
```

### 예시 D: ALD 공정 GPC 분석

```
1. x, y, thickness_nm, n_cycles 컬럼이 있는 CSV 준비
2. 사이드바에서 파일 선택 / 컬럼 매핑: X=x, Y=y, Data=thickness_nm
3. ⚗️ GPC 탭 → 두께 컬럼: thickness_nm / 모드: 컬럼 / 사이클 컬럼: n_cycles
4. GPC Heatmap 및 Center/Mid/Edge 통계 확인
5. 📄 보고서 탭 → GPC 포함 체크 → Excel 보고서 다운로드
```

### 예시 E: 결함 위치와 공정 데이터 상관 분석

```
1. 두께 측정 CSV + 결함 좌표 CSV 두 파일 준비
2. 단일 분석 → 두께 CSV 로드 → Heatmap 확인
3. 🔍 결함 오버레이 탭 → 결함 파일 선택 → 클래스 선택 (예: Particle, Scratch)
4. 두께 낮은 영역과 결함 분포의 공간적 상관관계 분석
```

### 예시 F: 로트 내 이상 웨이퍼 자동 탐지

```
1. 동일 공정 웨이퍼 10장의 측정 CSV 준비
2. 비교 분석 서브탭에서 10개 파일 등록 (나란히 비교도 가능)
3. 🤖 ML 탭 → 🔄 앱 동기화 → 10개 데이터셋 자동 불러오기
4. Contamination 0.10, 해상도 40 → 이상 탐지 실행
5. PCA 산점도에서 군집 이탈 웨이퍼 확인
```
